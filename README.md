# Motion Pipe

![Motion Pipe Banner](./apps/web/public/logo.png)

**The AI-Powered Motion Design Infrastructure.**

Motion Pipe is an autonomous AI agent for generating broadcast-ready motion graphics and product advertisements. It orchestrates Google Veo 3, ElevenLabs, and FFmpeg to transform text briefs into high-fidelity video assets.

## Features

*   **AI Video Generation:** Powered by Google Veo 3.
*   **Professional Voiceover:** ElevenLabs integration for natural narration.
*   **Escrow Payments:** Secure project milestones with Paystack Escrow.
*   **Asset Discovery:** Automated Selenium bots to find brand assets.
*   **Real-time Updates:** WebSocket streaming for live generation feedback.

##  Tech Stack

*   **Frontend:** Next.js 14, React, Tailwind CSS, Lucide React.
*   **Backend:** Golang (Gin/Fiber), Gorilla WebSockets.
*   **Video Processing:** FFmpeg, Google Veo 3 API.
*   **Storage:** Cloudflare R2 (S3 Compatible).
*   **Database:** PostgreSQL / Redis.
